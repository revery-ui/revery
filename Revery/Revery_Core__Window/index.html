val setSize : width:int -> height:int -> t -> unit

setSize(~width, ~height, window) sets the window size, taking display scaling into account.

For example, for a Windows display with 200% scaling, setSize(~width=400, ~height=300, window), will actually set the window to a height of 800 pixels wide by 600 pixels high. This is usually the behavior you want. To directly set the size, without considering display scaling, use setUnscaledSize
